# Attack phase

Attack phase as defined by DataVolley: either "Reception", "Transition
sideout" or "Transition breakpoint", assigned only to attack actions.

## Usage

``` r
dv_attack_phase(x)
```

## Arguments

- x:

  datavolleyplays: the plays component of a datavolley object as
  returned by [`dv_read()`](dv_read.md)

## Value

Character vector
